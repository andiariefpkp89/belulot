# Belulot

Welcome to Belulot! This project is a starting point for building amazing applications.

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/andiariefpkp89/belulot.git
cd belulot
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

### Running the Project

To start the development server:
```bash
npm run dev
# or
yarn dev
```
